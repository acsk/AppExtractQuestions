Pergunta 1Correto

Sua resposta está correta

Usar uma integração do API Gateway para publicar uma mensagem em um tópico do Amazon Simple Notification Service (Amazon SNS) quando o aplicativo receber um pedido. Inscrever uma função AWS Lambda no tópico para realizar o processamento.
Usar um autorizador do API Gateway para bloquear quaisquer solicitações enquanto o aplicativo processa um pedido.
